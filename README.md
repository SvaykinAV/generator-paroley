# Простой менеджер паролей

Этот проект помогает:
- Создавать надежные пароли
- Проверять, насколько пароль надежный
- Сохранять пароли в зашифрованном виде на вашем компьютере

## Как запустить

1. Установите Python 3.6 или новее
2. Откройте командную строку и перейдите в папку с проектом
3. Установите зависимости:
pip install -r requirements.txt
4. Запустите программу:
python password_helper.py
